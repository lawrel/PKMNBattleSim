# Pokemon Battle Simulator

## Goals  
+type matchups  
+status effects  
+weather effects  
+pokemon through gen 8  
+healing items  
+held item effects  
+navigable bag
+full party of pokemon

## Data
+pkmn data format: { name, species, number, "dex entry", type1, type2, height, weight, base hp, base attk, base spattk, def, spdef, speed, expyield, egggroup1, egggroup2, hatchcount, genderDistribution, evolution, evolution method, base friendship, | moveset , | tm moves , | egg moves  }  
+move format:  name, type, phys/spec/none, power, "description", status, weather   
+status effects will be hard coded  
+type matchups: type1, |weaknesses, |resistances, |ineffective
