# Pokemon Battle Simulator
Text based simulation of Pokemon battling. Read in pokemon, moveset, move, and item data from txt files. 
Intended to be the base for a larger project that hopefully one day has custom 2D Animation

## Goals  
+status effects  
+weather effects  
+pokemon through gen 8  
+healing items  
+held item effects  
+navigable bag
+held party of 6 pokemon  
+randomly generate enemy/wild pokemon from pokedex data & specified level range  
+abilities
+Alt Forme pokemon
+experience
+level up pokemon
+searchable pokedex

## COMPLETED
+type matchups  
+egg groups  
+breeding calculations  
+pokemon through gen 6 (Normal Forme only)  
+Nature information  
+basic move framework   

## ISSUES
There is a very unfortunate problem with moves and abilities - their effects are practically boundless, meaning many of them will need to be hardcoded (Trick Room). Most basic attacking and stat-boosting moves will avoid this issue.
